🚀 使用流程
================================================================================

步骤1: 安装依赖
   sudo apt-get install build-essential libpcap-dev

步骤2: 编译程序
   make

步骤3: 运行程序
   sudo ./ip_analyzer

步骤4: 选择网卡
   输入网卡索引号 (0-9)

步骤5: 查看结果
   程序自动捕获和解析IP包

步骤6: 停止程序
   按 Ctrl+C 停止捕获

================================================================================
📞 技术支持
================================================================================

问题1: 编译失败
   解决: 安装libpcap开发库
   命令: sudo apt-get install libpcap-dev

问题2: 权限不足
   解决: 使用sudo运行
   命令: sudo ./ip_analyzer

问题3: 找不到网卡
   解决: 检查网卡状态
   命令: ip link show

问题4: 捕获不到包
   解决: 生成测试流量
   命令: ping 8.8.8.8

问题5: 程序崩溃
   解决: 使用测试程序验证
   命令: make -f Makefile.test && ./test_packet_parser
